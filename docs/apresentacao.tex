\begin{frame}{Introdução e Escopo}
    \begin{itemize}
        \item \textbf{Objetivo:} Comparar a estrutura técnica de cardápios transregionais.
        \item \textbf{Fontes de Dados:}
            \begin{itemize}
                \item Camarões (Natal/RN): Parsing de PDF.
                \item Coco Bambu (Fortaleza/CE): Extração via DOM/Console JS.
            \end{itemize}
        \item \textbf{Tecnologias:} Python, NetworkX, Pyvis e Gemini (NER).
    \end{itemize}
\end{frame}

\begin{frame}{Metodologia: Projeção Monopartida}
    Além do grafo bipartido, aplicamos a \textbf{One-Mode Projection}:
    \begin{itemize}
        \item \textbf{Redução de Ruído:} Ingredientes são omitidos para focar na afinidade direta entre pratos.
        \item \textbf{Filtro de Significância:} Apenas conexões com Índice de Jaccard $J(u,v) \geq 0.50$ são mantidas.
        \item \textbf{Visualização:} Pratos similares funcionam como "ímãs" no layout de forças, evidenciando clusters de convergência.
    \end{itemize}
\end{frame}
